# Divisible Numbers Counter

## Overview
Divisible Numbers Counter is a C# Windows Forms application designed to help users quickly find numbers divisible by a specific term within a given range. The application provides a user-friendly interface with customizable text output, including color and font style options.

## Features
- Count numbers divisible by a user-selected term.
- Display results in a multiline textbox.
- Automatically split output into lines after every 10 numbers for readability.
- Choose text color via radio buttons (Black, Red, Blue, Green).
- Apply font styles: Bold and Italic (independently or together).
- Input validation to prevent errors (only numbers accepted in range fields).
- Simple exit button to close the application.

## Usage
1. Enter a starting number and an ending number.
2. Select a divisible term from the combo box.
3. Optionally, choose a text color and/or font style.
4. Click the **Count** button to display all divisible numbers in the textbox.

## Requirements
- Windows OS
- .NET Framework / .NET runtime compatible with Windows Forms
